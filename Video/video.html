Before starting this Content we must need to get to know that which type of video is better and supported to your browser if you don't then you need to know about encoding types Things that you need to keep in mind are:

1st is make sure that how your media is encoded( how media is encrypted in binary numbers algorithm) and the method which is used to encode the media is CODEC what it basically did is code/decode compress or decompress the media.
2nd thing you need to get to know that where or in which container this codec is stored. Basically you can see it like a Zip file which store the compressed files.
Some most popular media types (video) are‐
  1. MPEG‐4 container + H.264 video codec + AAC audio codec
    suffixes are .mp4 , .m4a
  2. WebM container + VP8 video codec + Vorbis audio codec
    suffixes are .webm
  3. WebM container + VP9 video codec + opus(libopus) audio codec
    suffixes are .webm
  4. ogg container + theora video codec + vorbis audio codec
    suffixes are .ogg and .ogv
  5. HEVC container + H.265 video codec + AAC audio codec
    suffixes are .hevc
These were the codecs that we'll use in this Webpage.
