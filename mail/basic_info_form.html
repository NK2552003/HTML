How to Make Form

Form Element

To make a Form in Webpage you need to inert the form element.
    <form> element.
it has both opening and closing Tag.

It has two main attributes

  1. Action
  2. Method

Action

Action attribute let's you insert the url of the application or script that will process the data and save it to server and give the response.
If you don't know how to make a script or application you can just keep it blankZJust for basic practice\.

Method

METHOD attribute send the request to the server which contains some special headers with url and post it or get it to the data.
  It has two types of methods:  
  1. POST
  2. GET

by default it is set to GET which is inappropriate to post the information. So, we'll use POST While making the form.
